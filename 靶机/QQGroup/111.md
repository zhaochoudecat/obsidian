**关键原理**：Pickle 反序列化 `wfuzz` 在加载文件时，使用的是 Python 的 **`pickle`** 模块。
- **序列化**: 把内存中的对象（比如代码、数据）变成二进制流保存到文件。
- **反序列化**: 把文件中的二进制流变回内存对象。
- **漏洞**: `pickle` 是不安全的。如果我们在文件中构造一段恶意的二进制流，当程序（以 root 权限）反序列化它时，就会自动执行我们在其中嵌入的恶意代码。

import pickle
import os

class RCE:
